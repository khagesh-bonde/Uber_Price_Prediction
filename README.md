# 🚖 Uber Price Prediction using Artificial Intelligence & Machine Learning

## 📊 Dataset
Uber Fares Dataset from Kaggle  
https://www.kaggle.com/datasets/yasserh/uber-fares-datase

---

## 🧠 Objective
To predict the fare amount of an Uber ride based on pickup and drop-off locations using Machine Learning models.

---

## ⚙️ Steps Performed
1. Data Preprocessing  
2. Outlier Detection and Removal  
3. Correlation Analysis  
4. Model Training  

---

## 🤖 Models Used
- Linear Regression  
- Random Forest Regression  

---

## 📈 Evaluation Metrics
- R2 Score  
- RMSE (Root Mean Squared Error)

---

## 🚀 Results
- Random Forest performed better than Linear Regression  
- Achieved good accuracy with lower error  

---

## 🛠 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 📌 Conclusion
Random Forest Regression is more effective for Uber fare prediction due to its ability to handle non-linear relationships.
